# TagAlong – Campus Meetup Platform

TagAlong is a web platform designed for university students to find or create small meetups on campus.

## Features
- Create campus meetups
- Discover events happening today
- Join other students
- Filter by category and gender preference
- Firebase authentication
- Firestore database

## Live Website
https://tagalong-amity.web.app

## Tech Stack
- HTML
- CSS
- JavaScript
- Firebase Authentication
- Firebase Firestore
- Firebase Hosting
